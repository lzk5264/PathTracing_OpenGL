# PathTracer-OpenGL

该项目的目标是实现一个路径追踪渲染器，支持各种模型，设置参数等

## 4/15/2024 进度、问题与后续安排

目前已经实现了支持理想漫反射材质的路径追踪

问题2： 进度慢，主要是在很多无意义的地方浪费了时间，比如模型导入和BVH构建完全可以找更加成熟的开源代码。

后续安排

1. 代码重构，使其更符合软件工程规范。
2. 找到开源的项目和代码来快速优化旧功能，实现新功能。
